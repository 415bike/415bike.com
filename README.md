415bike.com
===========

This is the open-source Sinatra app that powers 415bike.com

Improve it
----------

    git clone https://github.com/415bike/415bike.com.git
    cd 415bike.com
    bundle
    foreman start
    
Then open [localhost:5000](http://localhost:5000) in your browser. Huzzah!
